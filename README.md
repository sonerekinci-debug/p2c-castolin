# P2C - Tekliften Teslimata

Castolin Eutectic için hazırlanan, teklif sürecinden teslimata kadar tüm aşamaları takip eden web tabanlı yönetim sistemi.

## Özellikler
- **Dashboard**: Özet veriler ve aksiyon bekleyen işlemler.
- **Teklif Yönetimi**: Teklif oluşturma, revizyon takibi ve onay süreçleri.
- **Proje Takibi**: Onaylanan tekliflerin üretim, planlama ve sevkiyat süreçleri.
- **Departman Bazlı Yetkilendirme**: Satış, Teknik, Üretim, Kalite ve Muhasebe için özel görünümler.
- **Supabase Entegrasyonu**: Veri depolama ve kimlik doğrulama.

## Kurulum
Proje tek bir HTML dosyası (`p2c_v6.html`) üzerinden çalışmaktadır. Herhangi bir sunucu kurulumuna gerek duymadan tarayıcıda açılabilir.

## Teknolojiler
- HTML5 / CSS3
- JavaScript (Vanilla)
- Supabase (Backend as a Service)
